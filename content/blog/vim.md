 В этой заметке мы поговорим о базовых и дефолтных перемещениях в Вим  и Вимоподобных редакторах кода , зная которых облегчаете свою жизнь (может быть).
Эти команды  (шорткаты) помогут вам перемещаться между позициями курсора, поисковыми результатами, буферами и метками.

#### Перемещения по журналу:

Перейти назад по журналу переходов (jump list):
    
    ```
    Ctrl-o
    ```

Перейти вперёд по журналу переходов (jump list):
    
    ```
    Ctrl-i
    ```

Перейти к предыдущему точному месту (по последнему перемещению):
    
    ```
    ``
    ```

 Перейти на строку предыдущего места (по последнему перемещению):
    
    ```
    ''
    ```

#### Работа с метками (marks):

Перейти к глобальной метке (другой файл):
    Перемещает на строку с меткой `A-Z`.
    ```
    'A, 'B, ... 'Z
    ```
    
    
    
Перейти к локальной метке (текущий файл):
    Перемещает на строку с меткой `a-z` в текущем файле.
    ```
    'a, 'b, ... 'z
    ```
    

    
Перейти к точному месту локальной метки:
    Возвращает к точной позиции с меткой.
    ```
    `a, `b, ... `z
    ```
    

    
Сохранить метку в текущем месте:
    Сохраняет метку `a` в текущем положении курсора.
    ```
    ma
    ```


#### Переходы по поисковым результатам:

Перейти к следующему совпадению (вниз):
    
    ```
    n
    ```

Перейти к предыдущему совпадению (вверх):
    
    ```
    N
    ```

Перейти к следующему слову под курсором:
    
    ```
    *
    ```

Перейти к предыдущему слову под курсором:
    
    ```
    #
    ```


#### Переходы по тексту:

Перейти к первой строке файла:
    
    ```
    gg
    ```

Перейти к последней строке файла:
    
    ```
    G
    ```

Перейти на строку `N`:
    
    ```
    Ngg или :N
    ```

Перейти на строку с последним изменением:
    
    ```
    g;
    ```

Перейти на строку перед последним изменением:
    
    ```
    g,
    ```


#### Переходы между буферами и файлами:

Перейти к предыдущему файлу (альтернативный файл):
    
    ```
    Ctrl-^
    ```
    
Перейти к следующему буферу:
    
    ```
    :bnext (или :bn)
    ```
    
Перейти к предыдущему буферу:
    
    ```
    :bprevious (или :bp)
    ```
    
Открыть буфер по номеру:
    
    ```
    :bN
    ```
    
Где `N` — номер буфера.


#### Просмотр истории переходов:

Показать журнал переходов (jump list):
    
    ```
    :jumps
    ```
    
Очистить журнал переходов:
    
    ```
    :clearjumps
    ```
    
Показать метки (marks):
    
    ```
    :marks
    ```
    
Удалить метку:
    
    ```
    :delm a
    ```


#### Работа с быстрыми исправлениями и локациями:

Перейти к следующей позиции списка быстрого исправления:
    
    ```
    :cnext
    ```
    
Перейти к предыдущей позиции списка быстрого исправления:
    
    ```
    :cprevious
    ```
    
Перейти к следующей позиции списка локаций:
    
    ```
    :lnext
    ```
    
Перейти к предыдущей позиции списка локаций:
    
    ```
    :lprevious
    ```
